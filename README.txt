# Health Data Analysis Project

## Project Overview
This project aims to analyze the relationship between literacy rates in adult males and their knowledge about HIV across different countries. The dataset includes various indicators related to population, literacy, and HIV knowledge.

## Data
The dataset used for this analysis is sourced from `data.csv`. It contains records with various indicators related to health and demographics across multiple countries. Key indicators include literacy rates, HIV knowledge, urban and rural population statistics.

## Setup
To replicate this analysis, you will need:
- Python
- Jupyter Notebook
